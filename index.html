계정 삭제 안내 (Account Deletion Guide)

이 페이지는 Google Play Console의 요구사항에 맞춰, 사용자가 본인의 계정 및 연관 데이터를 삭제할 수 있는 방법을 안내합니다.

1. 계정 삭제 요청 방법

계정 삭제를 원하시면 아래 이메일로 요청해 주세요.

이메일: Pettogether98@naver.com

이메일에 반드시 포함해야 할 정보:

  • 앱에서 사용한 계정 ID 또는 가입 이메일
  • 본인 확인을 위한 기본 정보(예: 닉네임, 가입 날짜 등)
  • 제목 또는 본문에 계정 삭제 요청 또는 Account Deletion Request 명시

2. 처리 절차

  1. 요청 접수 및 본인 확인 절차 수행
  2. 삭제 가능한 데이터 확인 및 내부 검토
  3. 서버에서 사용자 계정 및 연관 데이터 삭제 수행
  4. 삭제 완료 시 안내 이메일 발송

처리 기간: 요청 접수 후 최대 7일 이내 처리됩니다.

3. 삭제되는 데이터

계정 삭제 시 아래 항목이 영구 삭제됩니다:

  • 사용자 프로필 정보(이름, 닉네임, 프로필 사진 등)
  • 앱 내 활동 데이터(게시물, 메시지, 기록 등)
  • 서버에 저장된 설정 및 개인화 정보
  • 기타 개인식별이 가능한 데이터

단, 관련 법령(회계·세무·보안 등)에 따라 일정 기간 보관해야 하는 데이터는 예외가 될 수 있으며, 해당 경우에는 별도 안내드립니다.

4. 유의사항

  • 계정 삭제는 되돌릴 수 없습니다. 삭제 완료 후 동일한 데이터 복구가 불가능합니다.
  • 삭제 요청 시 본인 확인을 위해 추가 정보를 요청할 수 있습니다.
  • 자동화된 계정 삭제 기능이 있는 경우, 해당 절차에 따라 즉시 삭제가 진행될 수 있습니다.
